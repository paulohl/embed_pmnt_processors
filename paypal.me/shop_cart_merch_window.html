\** Open shopping cart in the merchant window When a buyer clicks the Add to Cart button, the PayPal shopping cart opens in a separate browser window. You can use the target attribute in the form element to open the PayPal shopping cart in the same browser window that displays your website. Values for target attribute that open the PayPal shopping cart: | | | |-|-| | Target expression | Where cart opens | | target="paypal" | Separate browser window or browser tab from the merchant website. | | target="_self" | Same browser window or browser tab as the merchant website. | This sample opens the shopping cart in the same browser window that displays a merchant website. **\
